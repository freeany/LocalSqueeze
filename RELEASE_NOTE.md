- ⚙️ **更新版本至 1.0.2**：稳定性和性能提升

- 🔧 **隐藏快速操作区域**：优化用户界面，提供更清爽的操作体验

- 🎨 **优化页面样式**：改进整体视觉效果和用户交互

- 📚 **帮助页面**：全新的帮助文档，让您快速上手
- 📦 **Windows 安装程序支持**：提供更便捷的安装体验
- 🔄 **应用显示问题修复**：解决在不同分辨率下的显示异常

## 📥 安装方式

```bash
# 克隆仓库
git clone https://github.com/freeany/LocalSqueeze.git

# 安装依赖
npm install

# 启动应用
npm start
```

或直接下载最新的安装包：
